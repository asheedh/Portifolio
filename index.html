Junior Cybersecurity Analyst CISCO (Present)


A Junior Cyber Security Analyst is responsible for monitoring and protecting an organization's computer systems and networks. They assist in investigating security incidents, conducting vulnerability assessments, and applying necessary updates to prevent breaches. This role involves using security tools like SIEM and vulnerability scanners to identify and mitigate threats. Junior analysts also help ensure compliance with security policies and educate staff on best practices.
